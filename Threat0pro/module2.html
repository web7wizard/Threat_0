TRAINING MANUAL: MALWARE DEFENSE

1.0 DEFINITION

Malware (Malicious Software) is any software intentionally designed to cause damage to a computer, server, client, or computer network. It is the primary weapon in cyber warfare.

2.0 THREAT TYPES

  • TROJAN HORSE (Level 2.1)
    Disguised as legitimate software to trick users into installing it. Unlike viruses, Trojans do not replicate, but they open "backdoors" allowing attackers to control the system remotely.
  • WORM (Level 2.2)
    A standalone program that replicates itself to spread to other computers. It relies on security failures (open ports/unpatched vulnerabilities) to travel across networks without user interaction.
  • RANSOMWARE (Level 2.3)
    Malware that encrypts the victim's data and demands payment for the decryption key. Paying the ransom does not guarantee data recovery.

3.0 MODULE OBJECTIVES

  • Identify and terminate suspicious processes in Task Manager.
  • Utilize network isolation to stop the lateral spread of worms.
  • Execute Disaster Recovery (Backups) instead of paying ransoms.
